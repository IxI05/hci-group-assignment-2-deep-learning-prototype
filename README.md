# Farm2Energy Deep Learning Prototype

Farm2Energy is a Tkinter-based prototype that uses computer vision and a simple machine learning model to estimate livestock waste weight, predicted biogas yield, and estimated farmer revenue from a live camera feed or uploaded image.

## Features

- Live camera preview
- Local image upload
- Brown pixel segmentation with contour detection
- Linear regression prediction for waste weight
- Biogas and revenue estimates
- Voice/audio feedback after analysis

## Requirements

- Python 3
- A working camera for live capture
- Python packages listed in `requirements.txt`

## Run

On macOS, double-click:

```sh
run_app.command
```

Or run from the project folder:

```sh
python3 "HCI-images-deep learning/test_2.py"
```

## Install Dependencies

```sh
python3 -m pip install -r requirements.txt
```
